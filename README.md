Проект автоматизации тестирования некоторых функций интернет-магазина

Тесты написаны на языке Python с использованием Selenium и pytest, и паттерна Page Object Model.
Данный проект содержит автотесты для проверки функциональности интернет магазина, а именно:
- Добавление в корзину
- Регистрация нового пользователя
- Видимость элементов на каждой странице сайта, а именно Корзины и ссылки на регистрацию
- Переходы из различных страниц сайта в корзину и логин
